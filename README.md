# DJISDKExtension

Extension for DJISDK-iOS.

## Installation

DJISDKExtension is available through [CocoaPods](https://cocoapods.org). To install it, simply add the following line to your Podfile:

```ruby
pod 'DJISDKExtension'
```

## Requirements

iOS 9.0 +

DJI-SDK-iOS 4.7
